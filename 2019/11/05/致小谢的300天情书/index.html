致小谢的300天情书

发表于 2019-11-05 | 分类于 随笔 | | 阅读次数:
字数统计: 1,411 字 | 阅读时长 ≈ 5 分钟

亲爱的小谢：

今天是11月5日，现在是零点——我们在一起的第三个一百天。

回顾过去的300天，我的脑海中浮现出来几个词语——思念、争吵和我爱你。

思念

我们在一起的过程颇具戏剧性，和我的家人、同学、朋友们说起来大家都会很惊奇，“这都行？”是我听到最多的问题，我也在想，为什么这都行，为什么我就能认定就是你，然后在考研这段特殊的时间里坐在你的旁边将近三个月？为什么你在这三个月里就没有因为旁边一直出现同一个男生而离开那个位置？

不过我的搭讪也确实够烂的。第一次和你搭讪之前，我紧张了大半天，总是在担心在搭讪之后你会不会讨厌我然后离开那个位置，第一次和你说话是“同学，你觉得我的耳机声音大吗？”，现在想起来也觉得好傻，那个时候还带着写了个纸条问你，因为我担心自己会紧张的说不出话。

再后来就是强行帮你搬书，强行去接考完研的你。之后的那段时间，虽然考完试了，但是我们还是心照不宣的去了图书馆，还是原来的位置，就像一开始那样。

2019年的圣诞节，我拉着刚一起吃晚饭的室友们去逛商场，想给你准备一件特别的礼物，第一份礼物，最后居然选了一块毛巾，我也不知道当时在想什么，就觉得送礼物送毛巾的好像没听说过，一定会很特别！

记忆中的上一个冬天，好冷好冷，那时的南京真的好美，有雪，有阳光，有你。1月9号晚上，我们去吃了我最爱的江南里，晚上吃完饭之后去学校一号门对面的百货超市买了东西，那个晚上，我表白了。（其实我也不是那么喜欢走路啦，但是每次你都会陪着我走，所以每天吃完饭说什么也要拖着你逛一会儿啦）

之后的十天，每天都好像泡在蜜罐里一样。

直到放假的那天，我们在禄口机场抱了足足十分钟吧，我也第一次意识到物理的距离将会成为我们最大的障碍。

无论是寒假的那一个月，还是之后暑假的那一个多月，抑或是现在，此时此刻，我都很想你，很想。思念真的是一种让人很欲罢不能的东西，你说过，做一些其他的事情就不会那么想了，我也试过，不过好像确实没什么太大的用处。因为客观现实就是我们确确实实身隔半个中国，能做到的，也只有思念对方了。

不过还好，思念之后都是会有希望的，我们会相见，我们会遵守相互之间的承诺，我们也会一直深爱对方。

谢谢你，让我知道，思念也不是那样痛苦的。

争吵

这大概就是我们在一起之后的第二个主题词了吧？不一样的成长环境、不一样的思维方式、不一样的性格造就了不一样的我们，既然有了不一样，那就肯定会导致冲突。所以从第一个寒假开始，三天一小吵，五天一大吵几乎成了我们的日常。爱钻牛角尖的我遇上了没那么愿意服输的你，每次我都苦苦的和你讲道理，每次你也听到我的前言就知道我又要开始站在道德的制高点上指责你了。

不过我总是在想那个时候的你，或许没有想到，我愿意与你讲道理，是因为我急于改变你然后让我们相处的更舒服；那个时候的我，或许也没有想到，你想听到的也许不是各种大道理，而是一句轻柔的我爱你。

很多次，我们都近乎崩溃，在河海大学学生一舍的楼下，在重庆大学的校园外，在泰国的大巴上，在异地的电话里……争吵就一定是分开的前兆吗？我觉得不一定。有的时候，因为性格的迥异以及沟通方式的不同，在对方无法理解自己更深层次的意思的时候，就会爆发争吵。有的时候可能是我忘了一件对你来说很重要的事情，但是可能对我来说，我真的很健忘也不是故意不去做，但是你就会觉得我没有用心或者我根本没打算去做。这样的思维差异让我们吵了好多好多次，我也真真切切的认识到了错误，现在学会用手机提醒事项了，以后我会尽量不忘滴！

我爱你

像300天以前那样，喜欢你。
比300天以前那样，爱你。
喜欢 是每天黏在一起也不够
而爱 是即使异地也全都是你
喜欢 是我想吃你喜欢吃的所有东西
而爱 是我一定会学着亲手做出它们
喜欢 是试图去改变对方
而爱 是尝试去接纳对方
喜欢 是因为你，我的生活更加多彩
而爱 是因为我们，未来世界充满希望
我爱你，也谢谢你爱我。
