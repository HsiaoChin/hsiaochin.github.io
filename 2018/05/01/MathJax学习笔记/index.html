MathJax学习笔记

发表于 2018-05-01 | 分类于 学习笔记 | | 阅读次数:
字数统计: 1,301 字 | 阅读时长 ≈ 6 分钟

MathJax是一种可以在 浏览器完美显示数学公式和符号 的JavaScript库（？），渲染引擎很多，用它可以写出一些比较标准的公式，可以应用于许多场景。
我在刚接触这种语言时就被渲染之后的公式的高颜值所吸引，所以打算系统地学习一下，这也就是这篇笔记的由来。

本文由community wiki原创于Mathematics Meta。
由胡霄进翻译，转载请注明出处。

  1. 公式显示：行内公式与行外。
    将公式显示在与文字同行，应使用“$...$” 这个符号（注意$符号前后不应加入空格等），例如$\sum_{i=0}^n i^2 = \frac{(n^2+n)(2n+1)}{6}$在文档中的显示为$\sum_{i=0}^n i^2 = \frac{(n^2+n)(2n+1)}{6}$；而如果要将公式显示在空白区域，则应使用“$$...$$”，例如$$\sum_{i=0}^n i^2 = \frac{(n^2+n)(2n+1)}{6}$$将显示为$$\sum_{i=0}^n i^2 = \frac{(n^2+n)(2n+1)}{6}$$

  2. 希腊字母的显示。
    小写希腊字母\alpha,\beta,…,\omega显示为$\alpha$,$\beta$,…,$\omega$.（注意：在书写时都应加入$$符号）
    大写希腊字母\Gamma,\Delta,…,\Omega显示为$\Gamma$,$\Delta$,…,$\Omega$.

  3. 上标和下标的显示。
    上标使用^,下标使用_。例如：x_i^2显示为$x_i^2$.又如\log_2 x显示为$\log_2 x$.

  4. 集（这一部分标题可能翻译不准确）
    上下标只作用在紧跟其后的一个“集”上。一个“集”可以包含一个单独的符号或数字，也可以是由{...}包含的多个符号或数字。例如，如果输入10^10,则会显示出$10^10$这样奇怪的组合，这时应该输入10^{10},则会显示$10^{10}$,完美啊有木有？使用花括号可以指定上下标作用的区间：x^5^6会导致错误，显示KaTeX parse error: Double superscript at position 4: x^5^̲6，正确的写法是{x^y}^z或x^{y^z},这样都将显示${x^y}^z$.可以观察下如下两者的区别：x_i^2:$x_i^2$;x_{i^2}:$x_{i^2}$.

  5. 括号的显示。
    圆括号及方括号的输入方法不变，即()[]显示为$()$ $[]$,而花括号则需要加入\,即\{和\},显示为${}$.
    有时候会出现公式与括号的比例不协调的现象。比如:(\frac{\sqrt x}{y^3}):$(\frac{\sqrt x}{y^3})$，这样看好像括号偏小（虽然我觉得还行），但是加入\left(和\right)就会好一些，就像这样：\left(\frac{\sqrt x}{y^3}\right):$\left(\frac{\sqrt x}{y^3}\right)$.
    此外，\left和\right还可以用于所有括号，让它们变大（只有字面意思），此外，还有|: $|x|$, \vert :$|x|$, \Vert:$\Vert{x}\Vert$, \langle and \rangle :$\langle{x}\rangle$, \lceil and \rceil:$\lceil{x}\rceil$, and \lfloor and \rfloor:$\lfloor{x}\rfloor$.（最后这俩货是啥？）
    最后，手动调整括号大小的方法是：\Biggl(\biggl(\Bigl(\bigl((x)\bigr)\Bigr)\biggr)\Biggr):$\Biggl(\biggl(\Bigl(\bigl((x)\bigr)\Bigr)\biggr)\Biggr)$.

  6. 求和与积分。
    求和：\sum;积分：\int.下标是积分/求和下限，上标是上限。例如：\sum_1^n:$\sum_1^n$,此外，别忘了{}的奇用哦，比如：\sum_{i=0}^\infty i^2:$\sum_{i=0}^\infty i^2$.相似地，还有\prod:$\prod$(求积);\int:$\int$（积分）;\iint:$\iint$（二重积分）;\iiint:$\iiint$（三重积分）;\bigcup:$\bigcup$（集合取和）;\bigcap:$\bigcap$（集合取交）.

  1. 除法。
    有两种方法可以做到（frac和\over）。
    • \frac xy:$\frac xy$（适用于分子分母极其简单时）
  • \frac{a+1}{b+1}:$\frac{a+1}{b+1}$（适用于分子分母相对复杂时）
  • {a+1}\over{b+1}:${a+1}\over{b+1}$（适用于分子分母极其复杂时）
  1. 字体（好像没什么用）
  • 使用 \mathbb 或者 \Bbb f表示 “blackboard bold”字体: $\mathbb {CHNQRZCHNQRZ}$.
    • 使用 \mathbf表示 boldface字体: $\mathbf {ABCDEFGHIJKLMNOPQRSTUVWXYZABCDEFGHIJKLMNOPQRSTUVWXYZabcdefghijklmnopqrstuvwxyzabcdefghijklmnopqrstuvwxyz}$.
    • 使用 \mathtt 表示 “typewriter”字体: $\mathtt {ABCDEFGHIJKLMNOPQRSTUVWXYZABCDEFGHIJKLMNOPQRSTUVWXYZ abcdefghijklmnopqrstuvwxyzabcdefghijklmnopqrstuvwxyz}$.
    • 使用 \mathrm 表示罗马字体: $\mathrm {ABCDEFGHIJKLMNOPQRSTUVWXYZABCDEFGHIJKLMNOPQRSTUVWXYZabcdefghijklmnopqrstuvwxyzabcdefghijklmnopqrstuvwxyz}$.
    • 使用 \mathsf 表示无衬线字体（雾）: $\mathsf {ABCDEFGHIJKLMNOPQRSTUVWXYZABCDEFGHIJKLMNOPQRSTUVWXYZabcdefghijklmnopqrstuvwxyzabcdefghijklmnopqrstuvwxyz}$.
    • 使用 \mathcal 表示 “calligraphic” 字母: $\mathcal {ABCDEFGHIJKLMNOPQRSTUVWXYZABCDEFGHIJKLMNOPQRSTUVWXYZ}$
    • 使用 \mathscr 表示草稿字母: $\mathscr {ABCDEFGHIJKLMNOPQRSTUVWXYZABCDEFGHIJKLMNOPQRSTUVWXYZ}$
    • 使用 \mathfrak 表示 “Fraktur” (old German style)字母: $\mathfrak {ABCDEFGHIJKLMNOPQRSTUVWXYZabcdefghijklmnopqrstuvwxyzABCDEFGHIJKLMNOPQRSTUVWXYZabcdefghijklmnopqrstuvwxyz}$.

  1. 基♂符号或基数&指数（不知道翻译的对不对）。
    根号可以随着参数的变化而改变自己的大小，比如:\sqrt{x^3}:$\sqrt{x^3}$;\sqrt[3] {\frac xy}:$\sqrt[3] {\frac xy}$.而对于复杂的表达式，可以使用次方的形式，即{\frac xy}^{\frac 13}:${\frac xy}^{\frac 13}$.

  2. 一些诸如“lim”，“sin”，“max”，“ln”等特殊函数，使用\sin,lim等来将其字体转换成罗马体，就像这样：\sin x:$\sin x$，而不是sin:$sinx$.使用\lim时加入下标取极限区间:\lim_{x\to 0}:$$\lim_{x \to 0}$$
