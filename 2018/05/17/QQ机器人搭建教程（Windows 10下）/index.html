QQ机器人搭建教程（Windows 10下）

发表于 2018-05-17 | 分类于 计算机 | | 阅读次数:
字数统计: 1,197 字 | 阅读时长 ≈ 5 分钟
  • 之前一周都在复习钢筋混凝土结构，搞得我身心俱疲，考完之后逛GitHub无意中找到了一个比较好玩的项目，就是由大神写的一个QQbot，功能强大，不过想起我好像之前申请过一个QQ小号，就拿它来做实验咯。
  • 本文主要介绍如何具体搭建这个QQbot，功能就不具体介绍了，可以点这里来了解具体功能，我主要用这个QQbot（她的昵称是spirit，中文昵称没想好）来实现自动聊天，可以直接询问这个机器人一些比较简单的问题，省去了打开浏览器的麻烦，无聊的时候也可以调戏调戏这个机器人。

由于我所申请的AI借口有数量限制，所以请各位朋友不要与QQbot聊一些奇怪的问题（误）。昨天我一个智障高中同学–不愿透露姓名的熊博达同学，和机器人聊了半个多小时废话，对这个机器人百般刁难，甚至说出一些低俗的话，最后机器人都说不想聊了

教程开始

  1. 首先你的电脑里必须安装Python，具体安装过程：
  • 点击这个链接，进入下载页面，选择与你电脑兼容的可执行文件将其下载到你的电脑里
  • 找到你下载的exe文件，双击打开，进入安装页面，选择安装目录，一路next
  • 将Python添加到你的环境变量里，这个由于各个Windows版本不同添加方法有所不同，我的win10是1803版本，改变了一下进入方式，如果大家不知道如何添加，可以自行百度。
  1. 将qqbot这个package安装到我们的电脑里，具体过程：
  • 打开命令行：win+R ->输入cmd ->回车
  • 在命令行输入pip -v，查看pip管理包是否安装，如果已安装，则下一步
  • 输入pip install qqbot，等待

  1. 这个时候你的qqbot就可以使用了，但我的目标是运行自动回复脚本，所以接下来还有几步

  2. 到这个GitHub页面内点击右上角绿色download按钮下载这个文件

  3. 本地解压缩文件夹，打开文件夹，找到～/sample.py，把它复制粘贴到你的用户目录/.qqbot-tmp/plugins里面(即c:\user\xxx.qqbot-tmp\plugins)

  4. 右键 sample.py，edit该文件，把里面的代码全部删除，并把下面的代码全部拷贝到sample.py里

  5. 可以看到sample.py里有一个key，这个key就是图灵机器人API接口，我的key每天有限制，有需要使用的请联系我，方便我管理数据，欢迎大家自行注册并获取key（具体申请方法很简单，进入官网后按照指示即可，不过注册的时候有坑，就是虽然邮箱是非必须项，但不输入邮箱的话就无法获取验证码）。

  6. 一切就绪，现在：

  • 打开cmd，输入qqbot，系统会自动下载一个登陆二维码，用你需要变身机器人的QQ扫码登录
  • 再重新打开一个cmd，注意前一个不要退出，输入qq plug sample，这时候，就会自动加载sample插件了
  1. 成功了，可以开始实验咯（有的问题并不会精确回答，原因比较多，中文语义复杂，有的时候人工智能并不能精确识别，还有就是可能这个AI还比较弱，不同于Google AI之类有大量数据可以进行文本分析）

spirit之类的QQbot暂时不能识别图片，所以发送图片她只会接收到一段空字符，理解为你啥都没说

结语

是不是很简单呢？快试试吧，随文贴上spirit的QQ二维码，欢迎调戏（人工审核好友）
